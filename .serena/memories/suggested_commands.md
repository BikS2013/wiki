# Suggested Commands

## System Commands (Darwin/macOS)
- `git` - Version control
- `ls` - List directory contents
- `cd` - Change directory
- `grep` - Search text patterns
- `find` - Find files

## Node.js / TypeScript
- `npm init` - Initialize package.json
- `npm install` - Install dependencies
- `npx tsc --noEmit` - Type-check without emitting
- `npx tsx src/cli.ts` - Run TypeScript directly
- `npm run build` - Build project
- `npm test` - Run tests

## Development
- `npx tsx test_scripts/<test>.ts` - Run individual test scripts
