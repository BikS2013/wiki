# Task Completion Checklist

When a task is completed:
1. Run `npx tsc --noEmit` to verify TypeScript compilation
2. Run tests in test_scripts/
3. Check and update "Issues - Pending Items.md" - remove fixed issues
4. Update docs/design/project-design.md if design changed
5. Update CLAUDE.md if new tools were created
6. Do NOT perform version control operations unless explicitly requested
